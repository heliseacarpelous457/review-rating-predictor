## 🖥️ What this app does
review-rating-predictor is a Windows app that predicts product review ratings from written text. You paste or type a review, and the app estimates the rating based on the words used.

It uses two common machine learning parts:
- TF-IDF, which helps the app focus on important words
- Naive Bayes, which helps the app make a rating guess from text patterns

The app runs in a web page through Streamlit, so you use it in your browser after you start it on your computer.

## 🧠 How the model works
The app uses a simple text classification flow:
1. It turns review text into numbers with TF-IDF
2. It sends those numbers to a Naive Bayes model
3. The model returns a likely rating

This method works well for basic review prediction tasks and keeps the app fast and easy to run on a Windows computer.
